# Investment Policy Statement

## Purpose

Use the learning pool to build judgment, compare securities, and practice
disciplined decision-making. The goal is not short-term trading.

## Guardrails

- No automatic trading.
- Admin may suggest, compare, and warn, but Xiaosong makes final decisions.
- Every stock purchase should pass a trade checklist first.
- A new individual stock position should usually start small.
- One individual stock should normally not exceed 20% of the learning pool.
- If a single stock exceeds the soft limit, document the reason and exit rules.
- If a stock cannot be clearly compared against VTI, VOO, or QQQ, do not buy it.

## Preferred Recommendation Language

Use:

- Consider adding to watchlist
- Consider small position
- Continue observing
- Not attractive for now
- Consider trimming or avoiding

Avoid:

- Buy now
- Sell now
- Guaranteed return
- Must own
